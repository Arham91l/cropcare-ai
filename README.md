# 🌿 CropCare AI – Plant Disease Detection

This project uses Deep Learning (MobileNetV2) to detect plant diseases from leaf images.

## 🚀 Features

* Upload plant leaf image
* Detect disease or healthy condition
* Confidence score output
* Simple Gradio UI

## 🧠 Tech Stack

* TensorFlow / Keras
* Gradio
* NumPy, Pillow

## 📦 Installation

```bash
pip install -r requirements.txt
```

## ▶️ Run App

```bash
python app.py
```

## 📁 Files

* `app.py` → main application
* `plant_model.keras` → trained model
* `class_names.json` → class labels

## ⚠️ Note

Make sure TensorFlow version matches training environment.
